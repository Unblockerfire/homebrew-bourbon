# Homebrew Bourbon

Homebrew tap for **Bourbon**.

## Install

```bash
brew tap Unblockerfire/bourbon
brew install --cask bourbon
```

## About Bourbon

**Bourbon** is the modern continuation of the Whisky experience, updated for 2026 with a refreshed Wine runtime, improved compatibility, and active development.

Main project repo:

```text
https://github.com/Unblockerfire/Bourbon
```

## Notes

This tap contains the Homebrew cask definition only. Bourbon runtime binaries are distributed separately and are not stored in this repository.
